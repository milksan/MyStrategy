## 見出し

### 概要
`#` の数でレベルを指定する。H1〜H6まで対応。

### 構文

```markdown
# H1 見出し
## H2 見出し
### H3 見出し
#### H4 見出し
##### H5 見出し
###### H6 見出し
```

### 出力例

# H1 見出し
## H2 見出し
### H3 見出し
#### H4 見出し
##### H5 見出し
###### H6 見出し

### Props

| オプション | 構文 | 説明 |
|-----------|------|------|
| H1 | `#` | ページタイトル用。1ページに1つが推奨 |
| H2 | `##` | セクション区切り |
| H3〜H6 | `###`〜`######` | サブセクション |

## テキスト装飾

### 概要
文中の強調・装飾に使用するインライン記法。

### 構文

```markdown
**太字**
*イタリック*
~~打ち消し線~~
`インラインコード`
**_太字かつイタリック_**
```

### 出力例

**太字**
*イタリック*
~~打ち消し線~~
`インラインコード`
**_太字かつイタリック_**

### Props

| 記法 | 構文 | 説明 |
|------|------|------|
| 太字 | `**text**` または `__text__` | 強調 |
| イタリック | `*text*` または `_text_` | 斜体 |
| 打ち消し線 | `~~text~~` | 削除・非推奨を示す |
| インラインコード | `` `text` `` | コードや変数名 |
| 太字＋イタリック | `***text***` | 最強調 |

## リスト

### 概要
箇条書きリストと番号付きリストの2種類。ネストはインデント（スペース2〜4つ）で表現。

### 構文

```markdown
<!-- 箇条書き -->
- アイテム1
- アイテム2
  - ネストアイテム
    - 深いネスト

<!-- 番号付き -->
1. 最初
2. 次
3. 最後
   1. ネスト番号付き
```

### 出力例

**箇条書き**

- アイテム1
- アイテム2
  - ネストアイテム
    - 深いネスト

**番号付き**

1. 最初
2. 次
3. 最後
   1. ネスト番号付き

### Props

| 種類 | 記号 | 説明 |
|------|------|------|
| 箇条書き | `-` / `*` / `+` | 順不同リスト |
| 番号付き | `1.` `2.` ... | 順序あり。番号は自動採番される場合も |
| ネスト | インデント2〜4スペース | 階層構造 |

## リンク・画像

### 概要
外部リンクや画像の埋め込み。

### 構文

```markdown
<!-- リンク -->
[表示テキスト](https://example.com)
[表示テキスト](https://example.com "ツールチップ")

<!-- 参照スタイル -->
[表示テキスト][ref-id]
[ref-id]: https://example.com

<!-- 画像 -->
![代替テキスト](https://example.com/image.png)
![代替テキスト](./local/image.png "ツールチップ")
```

### 出力例

[表示テキスト](https://example.com)
[ツールチップ付きリンク](https://example.com "ここがツールチップ")

### Props

| 要素 | 構文 | 説明 |
|------|------|------|
| リンク | `[text](url)` | 通常リンク |
| ツールチップ | `[text](url "title")` | ホバー時に表示 |
| 参照スタイル | `[text][id]` + `[id]: url` | URLを文末で一元管理 |
| 画像 | `![alt](url)` | `!` を先頭に付ける |

## コード

### 概要
インラインコードとコードブロックの2種類。コードブロックには言語指定でシンタックスハイライトが適用される。

### 構文

````markdown
<!-- インライン -->
`const x = 1`

<!-- コードブロック（言語指定あり） -->
```javascript
const greet = (name) => {
  return `Hello, ${name}!`;
};
```

```python
def greet(name):
    return f"Hello, {name}!"
```
````

### 出力例

`const x = 1`

```javascript
const greet = (name) => {
  return `Hello, ${name}!`;
};
```

```python
def greet(name):
    return f"Hello, {name}!"
```

### Props

| 種類 | 構文 | 説明 |
|------|------|------|
| インライン | `` `code` `` | 文中のコード |
| コードブロック | ```` ```lang ```` | 複数行コード |
| 言語指定 | ```` ```js ````, ```` ```py ```` など | シンタックスハイライト |

## テーブル

### 概要
パイプ `|` で列を区切る。2行目でアライメントを指定する。

### 構文

```markdown
| 左揃え | 中央揃え | 右揃え |
|:-------|:--------:|-------:|
| Left   | Center   | Right  |
| 長いテキスト | 中 | 1 |
```

### 出力例

| 左揃え | 中央揃え | 右揃え |
|:-------|:--------:|-------:|
| Left   | Center   | Right  |
| 長いテキスト | 中 | 1 |

### Props

| 記法 | 意味 |
|------|------|
| `|---|` | デフォルト（左揃え） |
| `|:---|` | 左揃え |
| `|:---:|` | 中央揃え |
| `|---:|` | 右揃え |

## 引用

### 概要
`>` で始める引用ブロック。ネストや他の記法と組み合わせ可能。

### 構文

```markdown
> 引用テキスト

> 複数行の引用は
> 各行に `>` を付ける

> ネスト引用
>> 2段目
>>> 3段目

> **太字**や`コード`も使える
```

### 出力例

> 引用テキスト

> 複数行の引用は
> 各行に `>` を付ける

> ネスト引用
>> 2段目
>>> 3段目

> **太字**や`コード`も使える

## タスクリスト

### 概要
GitHubなど一部のレンダラーで対応するチェックボックスリスト。

### 構文

```markdown
- [x] 完了したタスク
- [ ] 未完了のタスク
- [x] もう一つ完了
  - [ ] ネストされた未完了
```

### 出力例

- [x] 完了したタスク
- [ ] 未完了のタスク
- [x] もう一つ完了
  - [ ] ネストされた未完了

### Props

| 記法 | 説明 |
|------|------|
| `- [x]` | チェック済み |
| `- [ ]` | 未チェック（スペース必須） |
