# 鉄拳8 シーズン3（Ver.3.00）調整まとめ

  

このファイルは、配信内で語られた内容をもとにした「方針ベース」のまとめです。  

細かいフレームや全キャラ個別の全項目ではなく、「ゲーム全体で何が変わったか」「代表的なキャラがどう変化したか」に焦点を当てています。

  

---

  

## 1. 全体方針・システム面の変更

  

### 1-1. 調整ボリュームについて

  

- シーズン2は「調整量が多すぎて、プレイヤーが追いつけないレベル」だったという認識。

- シーズン3は、**明らかに調整項目の量が減っている**。

  - 1キャラあたりの「性能調整」は概ね 5〜6 個前後が目安。

  - シーズン2のような「1キャラだけで数ページ」「全員大改造」という調整は避けられている。

- 開発側も「シーズン2はやりすぎだった」と考えているだろう、というニュアンスのコメントあり。

- 今後の標準は「このくらいの調整量」がベースになるのでは、という見立て。

  

### 1-2. ヒート／強化アイコン周りの共通変更

  

特に大きいのがここ。

  

- **「強化アイコン（キャラ固有強化）」持ちキャラが、ヒート終了後も強化状態を維持する仕様が“基本的に”削除**。

  - シーズン2までは：

    - ヒート発動 → 強化状態に入る → ヒートが切れても強化だけ残るキャラが多数。

  - シーズン3では：

    - ヒートが終わると、強化状態もほぼリセットされるようになった。

- その代わりに：

  - 強化を**別ルートで獲得できる新技**や派生が複数のキャラに追加されている。

    - 例：アスカ、クラウディオ、カムラムなどは、ヒート以外の行動から強化状態に入れる技が追加／変更。

  

この結果として：

  

- 「とりあえずヒート吐けば強化状態」という簡単ルートが減少。

- 強化状態に入るには、**明確なリスク行動や条件（特定の技ヒット／構えなど）**を通す必要が増えた。

- 強化を前提にした「高火力コンボ」や「長時間強化状態を維持した立ち回り」のパワーが全体に落ちている。

  

### 1-3. ヒートスマッシュと壁周り

  

- 多くのキャラで、

  - **ヒートスマッシュが「壁やられ／壁強」を誘発しなくなった**。

- 特に壁コンフィニッシュをヒートスマッシュに大きく依存していたキャラ（例：平八、ファラン、ブライアン等）は、火力とリターンが体感で分かるレベルで低下。

- 一部キャラは、ヒートスマッシュのヒートゲージ消費量も増加（例：ブライアンのネオインシネレーター、ネオマッハ等）。

  

まとめると：

  

- 「ヒートスマッシュぶっぱで壁から試合が終わる」パターンを全体として弱め、

- コンボルートやステージギミック活用を **より工夫してダメージを出す方向** に寄せている。

  

### 1-4. スパイク系コンボ・床壊しの調整

  

- 叩きつけ（スパイク）からのコンボや、床破壊ルートのダメージが複数キャラで調整されている。

  - 例：フェン、カムラム、クラウディオなど。

- 共通の傾向として：

  - **コンボの総ダメージを少し抑える方向**。

  - ただしステージギミック自体は引き続き強力で、新技追加でむしろ「狙いやすくなった」キャラもいる（ファランなど）。

## 2. 調整の全体的な特徴

  

### 2-1. 「キャラ差を縮める」方向

  

配信の中でも何度も触れられているが、シーズン3のバランス方針は：

  

- **極端に強い要素・理不尽な崩し・過剰な火力を削る**

- ただしそのキャラの「コンセプト」や「長所」はなるべく残す

- 結果として、「キャラ差が少し縮まる」方向

  

と受け取れる内容になっている。

  

具体的には：

  

- シーズン2で「明らかに頭ひとつ抜けている」キャラの要素に強めのナーフ。

- 逆に、明らかに不安定だったキャラは、

  - コンボの安定性向上

  - 理不尽な弱点（謎のスカり方・背面当たりなど）の修正

  - 確定反撃の強化

  などで底上げ。

  

### 2-2. 新技の方向性

  

配信者のコメントでも繰り返されているが、シーズン2と比べたとき：

  

- シーズン2の新技：  

  「もらった瞬間から“ヤバい”と分かるレベルの強技」が多かった。

- シーズン3の新技：

  - **渋い・地味・玄人向け** のものが多い。

  - 例：

    - アーマーキング：冗談潜り＋カウンターコンボはあるが、即壊れではない中段。

    - アンナ：牽制下段＋運び用コンボパーツなど、使い道はあるがインパクトは控えめ。

    - アスカ：横移動潰しの新中段（もみじおろし）は強いが、発生やリターンは過剰ではない。

    - ドラグノフ：スキャバードとの連携強化など、使い手が工夫して活かすタイプ。

- 新モーション自体も、完全新規より「既存技のモーション流用」が目立つ（開発コストを抑えた調整型アップデートの印象）。

## 3. 代表的なキャラ調整の要約

  

※配信内で特に詳しく触れられていたキャラのみ抜粋。  

※「かなり弱体化／弱体化／微弱体化／ほぼ変わらず／微強化／強化」などは配信者の主観込みの評価。

  

### 3-1. 明確な弱体化が大きいキャラ

  

#### アンナ

  

- 主な弱体化ポイント：

  - 右パンCH → キューピットアローなど、**各種ジュフレ（しゃがみ崩れ）ルートが軒並み削除**。

  - 左アッパー派生（アパスト〜ハイキック）のカウンターヒット時挙動変更で、

    - 壁強・追撃ルートが弱体化。

  - ミストレスヒール（下段〜キューピットアロー）もカウンター挙動変更。

  - 66RPヒット後の置き攻め弱体化（追撃確定が怪しくなった）。

  - しゃがみ中3LP / 3RP（コールドブレードなど）の横回避耐性が下げられ、スカりやすく。

  - キューピットアローのスカ硬直増加 → **見てから15Fクラスの技で確反が入るレベル**に。

- まだ強い点：

  - しゃがみ2択（中段／コールドブレード系）自体のダメージはそこまで落ちていない。

  - オープンセサミ、アルテミスなど、依然として優秀な技はそのまま。

- 評価：**かなり弱体化**。  

  ただし「最弱」とまでは言えず、武器技とコールドブレード周りでまだ十分戦える。

  

#### アスカ

  

- 主な弱体化：

  - 鬼殺しヒット後のやられ変更：

    - 以前のような「鬼殺し確定 → 置き攻めが強すぎる」状況が大きく弱体化。

    - 差しろ（差し返し）からの鬼殺しルートも弱く。

  - み月（しゃがみからの溜め中段）の発生が3Fほど遅くなり、削りダメージも半減。

    - 立ち回りでの「とりあえずみ月」がしづらく。

  - 壁コン火力・ヒート中の追撃でのダメージ低下。

  - ヒートバースト終了後の強化状態維持がなくなり、**ヒート後の継続バフが消失**。

- まだ強い点：

  - 壁際の総合力（差しろ・各種中段・構え）は依然高水準。

  - み月自体はまだ機能するが、「試合を決める一撃」から「1つの択」に格下げされたイメージ。

- 評価：**2段階くらいの弱体化**（シーズン2最強格 → 依然強いが“盛りすぎ”部分を削られた）。

  

#### ブライアン

  

- 主な弱体化：

  - ネオインシネレーター／ネオマッハ：

    - ヒートゲージ消費量が大幅増加。

    - コンボ中に両方使うとヒートがほぼ空になるため、**ヒート中の爆発力がかなり抑えられた**。

  - 挑発ヒートスマッシュ：

    - 専用の誘導時間が削除され、**挑発 → ヒートスマッシュがジャスト要求＆難化**。

    - 挑発→レイジアーツ等の距離も調整され、旧来の“ブライアン専用芸当”が弱体化。

  - 2Kガード後の距離調整：

    - 以前より距離が近くなり、確反・暴れのリスクが増加。

  - スネークコンボ（2RP RP RP RPLK系）の出し切りに確反がつくなど、固め連携にもスキ発生。

  - 強化アイコンがヒート後に残らない → 強化中段・強化下段の回転率低下。

- 強化・緩和点：

  - 新やられ（バイン系）追加による床ギミック活用ルートの開拓。

  - 一部技のヒット時距離調整でコンボ安定。

- 評価：**2段階弱体化**。  

  まだ強キャラだが、「とりあえずネオマッハ＋インシネレーター＋挑発」が通らなくなった。

  

#### クラウディオ

  

- 主な弱体化：

  - ディスペルバースト（4LP〜派生）の出し切りガードが **マイナ14** に。

    - 多くのキャラが浮き確定、ヒート発動も確定しやすくなった。

  - QRP（ホーミング中段）の横への強さと持続調整により、少し弱体化。

  - ブルームーン（アイコン消費技）関連：

    - ヒート終了後に強化アイコンを残せなくなり、**ヒート中のアイコン＋ヒート後ブルームーンの連続展開が難化**。

- 強化・新要素：

  - 新技「オービットスライス（7RP）」追加：

    - ヒート中はヒットから追撃可能。

    - スターバースト獲得ルートが増えた。

  - レイジアーツヒットでアイコン獲得など、新しい強化ルートもあり。

- 評価：**弱体化〜かなり弱体化の中間**。  

  アイコン周りの仕様変更の影響が重く、「ブルームーン前提」の設計が抑えられた。

  

#### クライブ（FF16）

  

- 主な弱体化：

  - フェニックスシフト〜4RP（構え派生）のガード後状況が悪化。

    - フェニックスシフト〜パワクラ読み合いに明確なリスク。

  - フェニックスシフト中の両P（パワクラ）の横追い性能やスカ硬直の増加。

  - しゃがみからの3RP（下段）など、**主要下段の確反が重くなったものが複数**。

  - 残鉄拳ゲージレベル3時の各種技ダメージ減少（48→43、80→75など）：

    - 残鉄拳を絡めたコンボの総ダメージが目に見えて低下。

- 強化・新要素：

  - 残鉄拳ゲージ自体は溜まりやすくなっている（通常技ヒット時のゲージ増加量アップ）。

  - 新技「ウィケットホイール」追加（7RP〜RP）：距離離し＋トルネード。

- 評価：**2段階弱体化**。  

  「残鉄拳を使え」というメッセージはあるが、総合的には素直に弱くされている。

### 3-2. 微弱体化〜ほぼ据え置きのキャラ

  

#### フェン

  

- 弱体化：

  - ヒートスマッシュの壁強削除 → スパイクコンボ火力ダウン。

  - 撲面（後ろRP）の“謎の避け”（フレカウンターを撲面で避けるような挙動）が修正。

  - 1RP RP RP の出し切りガードに**マイナ14相当の確反**が付くなど、固めのリスク増加。

- 強化・新要素：

  - 新技 7両P（ホールド中段）追加：タメでガード有利・床壊し・カウンターで新やられ。

  - 背向けから運手へ移行できる新技（背向け両P）追加。

  - 運手中の行動やコンボルートが細かく改善。

- 評価：**1段階弱体化寄り〜トントン**。  

  便利技にリスクがついた反面、運手周りの表現力が増えた。

  

#### ドラグノフ

  

- 強化点：

  - スキャバード（前前両P）の発生・リーチ強化 → スカ確・ヘルスラ連携が強力に。

  - 新技 7LP（ヘッドハンター系の右横潰し冗談）追加。

  - ヘルスラ関連の挙動安定（1発目のみヒット時の追撃など）。

- 弱体化：

  - ヘッドハンター（2発目）の空振り硬直増加 → スカ確を入れやすく。

  - 一部投げやフェイントキャッチの抜け猶予調整。

- 評価：**微強化**。  

  スキャバードと新7LPで、総合的には強くなったと見て問題ない。

  

#### カムラム

  

- 強化点：

  - 2Kカウンターヒットでの強化状態獲得。

    - 2K CH → 強化アイコン → その後の攻め・火力が上がる。

  - 66RK派生など、一部連携カウンターで新しい床壊しやコンボルート。

- 弱体化：

  - 強化アイコンがヒート終了で残らないため、シーズン2のような「常時強化状態」が減少。

  - QLK（置き中段）の弱体化（下方向判定縮小など）。

- 評価：**ほぼ変わらず〜微強化**。  

  2K強化はかなり強いが、ヒートシステム共通のナーフで帳尻を取られている。

  

#### カムラム以外の「強化アイコン持ち」全般

  

- 共通：

  - ヒート後強化維持削除は全員痛い。

  - その代わり、強化獲得技が増えたり、強化中技の調整が入ったりしている。

- 評価：

  - 「システム的には弱体化」だが、キャラごとの個別調整次第で体感は変わる。

### 3-3. 微強化〜明確な強化寄りのキャラ

  

#### アリサ

  

- 強化点：

  - デストロイフォームへの移行ルートが大幅増加。

    - 7LP（新技）・RK〜派生・後ろRPRK〜派生などから構え移行可能。

  - 新技 7LP：

    - ダウンに当たる中段＋デストロイフォーム移行。

    - 起き攻め・ダウン追いの選択肢として優秀。

  - 壁コン用の RP RP の挙動修正で、コンボ後の状況が安定。

  - ヒートスマッシュの「どこかに飛んでいく」挙動が改善され、受け側が最大反撃を入れやすい、健全な仕様に。

- 弱体化：

  - デストロイフォーム中ヒートスマッシュの暴れ部分（空振り時の移行）が削られた程度で、致命的ではない。

- 評価：**強化寄り**。  

  シーズン2から据え置き以上で、シーズン3でも上位候補として名前が挙がりそう。

  

#### デビル仁

  

- 強化点：

  - 確反強化：

    - 立ち途中LPRK（旧・十字斬系の中段）追加で 13F確反が強化。

    - ワンツー確反からのリターンも上がる。

  - コンボ安定：

    - 8両P（トルネード）の下方向判定強化 → 空コンでの拾い安定。

    - 3両P〜6L系のコンボルートなど、複数ルートで安定性アップ。

  - QLPの回復可能ダメージ廃止 → 実ダメージ化。

- 弱体化：

  - システム面のヒート・強化仕様以外には、目立った直接ナーフは少なめ。

- 評価：**1段階強化**。  

  操作難度は据え置きだが、真面目に使うリターンが増えた。

  

#### ドラグノフ（再掲）

  

- 前述の通り、微強化と見て良い。

  

#### アズセナ

  

- 強化点：

  - 横移動RK（新下段）追加：しゃがみステータスで相手の横移動を咎めつつ攻め継続。

  - R（構え中）や 6LK に**片側ホーミング性能**が付与され、横移動への対応力が向上。

  - 両Pの置き攻め性能やスラッシュ派生の当たり方が改善され、攻めのルートが分かりやすくなった。

- 評価：**1段階強化**。  

  元々トリッキーなキャラだが、横と置き攻めの信頼度が上がった。

## 4. シーズン3環境のざっくりした見通し

  

配信者のコメントと内容から読み取れる「シーズン3のゲーム像」はおおよそ次のとおり。

  

- **シーズン2で暴れていた“明らかに強すぎる要素”はかなり整理された。**

  - アンナ、アスカ、ブライアン、クラウディオ、クライブなどが代表。

- 代わりに：

  - アリサ、アズセナ、デビル仁、ドラグノフなど、**地力のあるキャラの堅実な強化**が目立つ。

- ヒート・強化アイコン周りの仕様変更により、

  - **「ヒート吐き得」から「ヒート中にどうリターンを取るか」に比重がシフト**。

  - 強化維持に頼るキャラは、立ち回りの組み立て直しが必要。

- 新技は総じて「渋く」「覚える量も控えめ」で、

  - シーズン2のような「パッチノートを読みきる前に環境が変わる」ほどの変動はなさそう。

  - その分、**競技シーン・長期的プレイにとっては健全なアップデート**と言える内容。
